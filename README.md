# Bone Appetite

This website has multiple navigation tabs that navigates through the respective pages. The home screen include the Logo with a catchy icon and navigation bar which can be navigated to contact us, about, registration and order page. The body consist of an image on the left of the screen with text aligned on the right describing the exceptional pizza of the shop. At the bottom of the screen is the footer ontains a copyright notice, contact information and social media icons. When a user clicks on the "Contact" link contact us page displays all contact information. On click of "About" the user is directed to an online italian magazine. The "Register" button allows user to register themselves for various benefits offered by the organization. This page include javaScript validation. "Order" button provides user an uncomplicated way to place there order.
